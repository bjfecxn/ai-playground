# AI Playground

这是一个收集测试 AI 过程中顺带生成的有趣静态网站的合集。

## 项目列表

### 中国历史全景
穿越五千年华夏文明，探索朝代更迭、文化传承与历史脉络的交互式可视化体验。

## 说明

这些项目都是在与各种 AI 工具交互、测试其能力时产生的副产品。虽然是"顺手"生成的，但每个项目都具有一定的趣味性和可玩性。

## 本地运行

由于都是纯静态网站，直接用浏览器打开 `index.html` 即可，或使用任意静态服务器：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve
```
